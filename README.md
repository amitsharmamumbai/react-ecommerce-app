## вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/amitsharmamumbai/react-ecommerce-app
```

Navigate to the project

```bash
cd ecommerce-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
